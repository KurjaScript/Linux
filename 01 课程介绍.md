服务器的操作系统绝大多数都是 Linux

### 学习 Linux 可以做什么

- 运维工程师：系统运维、应用运维、DBA

- 开发工程师：内核/驱动开发、软件开发、嵌入式开发

### 课程内容

1. 入门篇
   - Linux 概述
   - Linux 安装
2. 基础篇
   - 文件与目录结构
   - VI / VIM 编辑器
   - 网络配置
   - 系统管理
   - 远程登录
3. 实操篇
   - 文件目录类
   - 时间日期类
   - 用户管理类
   - 文件权限类
   - 磁盘分区类
   - 进程 线程类
4. 扩展篇
   - 软件包管理
   - 克隆虚拟机
   - Shell 编程
